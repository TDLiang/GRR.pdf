2. Going-up and Going-down  
Theorem 2.1. Suppose f : A ,→ B is an integral extension, then the induced  
scheme morphism f∗ : Spec B → Spec A is surjective.  
Theorem 2.2 (Going-up). Let Aꢀ B be two integral domain and f : A ,→ B be an  
integral extension. For any two prime ideals p1 ⊂ p2 in A and a prime ideal q1 in  
B such that q1 ∩ A = p1, then there exists q2 in B such that q2 ∩ A = p2.  
Going-down property: Let A ,→ B be an integral extension, p2 ⊂ p1 ⊂ A and  
q1 ⊂ B be prime ideals such that p1 ∩ A = p1, then there exists q2 ∈ Spec B such  
that q2 ∩ A = p2.  
Theorem 2.3 (Going-down). When A ⊂ B are rings and A is integrally closed,  
then the going-down property holds and the induced morphism f∗ : Spec B → Spec A  
is an open map.
